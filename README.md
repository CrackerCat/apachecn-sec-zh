## 下载

### Docker

```
docker pull apachecn0/apachecn-sec-zh
docker run -tid -p <port>:80 apachecn0/apachecn-sec-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-sec-zh
apachecn-sec-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-sec-zh
apachecn-sec-zh <port>
# 访问 http://localhost:{port} 查看文档
```
